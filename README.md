# 🩸 Blood Bank Management System

This repository contains a **Blood Bank Management System** developed as part of my **Micro Project**. The project is designed to simplify the management of blood donor records through a user-friendly web interface.

## 🎯 Project Objectives

- Develop a web-based blood bank management system.
- Manage donor information efficiently.
- Perform Create, Read, Update, and Delete (CRUD) operations.
- Understand database connectivity using PHP and MySQL.
- Strengthen web development fundamentals.

## ✨ Features

- 🔐 User Login
- ➕ Add Donor Records
- 👀 View Donor Details
- ✏️ Edit Existing Records
- ❌ Delete Records
- 💾 Database Connectivity

## 🛠️ Technologies Used

- PHP
- MySQL
- HTML
- CSS

## 📂 Project Structure

- `index.php` – Home Page
- `login.php` – User Authentication
- `db.php` – Database Connection
- `public_view.php` – Display Records
- `edit.php` – Update Records
- `delete.php` – Delete Records
- `logout.php` – Logout Functionality
- `style.css` – User Interface Styling

## 📚 Learning Outcomes

- PHP Programming
- MySQL Database Integration
- CRUD Operations
- User Authentication
- Frontend Styling with CSS
- Web Application Development

## 📌 Project Purpose

This micro project was developed to gain hands-on experience in full-stack web development using **PHP**, **MySQL**, and **CSS**, while implementing core database operations in a real-world application.
